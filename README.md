# Professional Certifications

Public record of verified professional certifications and structured learning
evidence across CRM, ERP, business analysis, and automation-adjacent domains.

## Coverage

This repository documents expertise in:

- Bitrix24 CRM and automation
- 1C ERP and system integration
- Business and system analysis
- Business process modeling (BPMN, eEPC)

## Why It Exists

- Supports public professional profiles with verifiable evidence.
- Complements product and automation repositories with domain credibility.
- Keeps certifications grouped by capability area instead of buried in a CV.

## Public Use

The repository acts as a supporting evidence base for GitHub, Notion, LinkedIn,
and portfolio-style project presentations.
